# Datasets for Sports Injury Risk Detection

## 1. COCO Keypoints Dataset
- Purpose: Pose estimation training, human motion analysis
- Source: https://cocodataset.org
- Status: To be integrated in Milestone 2

## 2. MediaPipe Pose (Built-in)
- Purpose: Real-time joint tracking and skeleton generation
- Source: Built into MediaPipe library by Google
- Status: Will be used directly in pose estimation engine

## 3. Human3.6M Dataset
- Purpose: Human pose estimation, joint tracking, movement analysis
- Source: http://vision.imar.ro/human3.6m
- Status: Reference dataset for Milestone 2

## 4. MPII Human Pose Dataset
- Purpose: Body keypoint detection, activity recognition
- Source: http://human-pose.mpi-inf.mpg.de
- Status: Reference dataset for Milestone 2
